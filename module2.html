Network and Information Security Management

The subjects covered in this module are the following:

- Fundamentals of computer networks and security.
- Principles of information security management.
- Use and benefits of various monitoring and logging tools.
- How to create and use security vulnerability and assessment tools.
- Future network architectures and information security management.
