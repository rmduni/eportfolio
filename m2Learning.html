Learning Outcomes

The objectives set for this module are for the students be able to:

  1. Identify and analyse security risks and vulnerabilities in IT network systems and determine appropriate methodologies, tools and techniques to manage and/or solve them.
  2. Design and critically appraise computer programs and systems to produce solutions that help manage and audit risk and security issues.
  3. Gather and synthesise information from multiple sources (including Internet security alerts and warning sites) to aid in the systematic analysis of security breaches and issues.
  4. Articulate the legal, social, ethical and professional issues faced by information security professionals.
This module has especially helped me to expand my technical knowledge in fundamentals of IT networks which is an essential requisite for Information Security professionals. The reading materials provided at the beginning of the module and the network scan exercise helped enormously in achieving this knowledge. Also, to describe the findings of the network scan in the network information security assessment report allowed to evaluate the state of a whole system environment. This covers the learning objectives 1, 2 and 3.

All the work done with standards and regulations during the last part of the module helped us to gain knowledge base in this area. This knowledge is necessary for professionals to confidently express and advice on legal requirements and implications inherited from the standards. The vast amount of information available in this area, requires continuous learning. It is necessary for professionals to expand knowledge in this area to be in a better position to articulate legal issues which is in line with the learning objective 4.
